# Poll Question Generator

Una herramienta automatizada en VBA para convertir hojas de cálculo de Excel en archivos CSV formateados y generar imágenes de preguntas automáticamente.

## Características
- **Exportación masiva:** Convierte columnas específicas de Excel a formato CSV (UTF-8).
- **Generación de imágenes:** Crea imágenes dinámicas ajustadas al tamaño del texto.
- **Configurable:** Rutas dinámicas para evitar errores de ubicación.

## Instalación
1. Descarga el archivo `.xlsm`.
2. Habilita las macros en Excel (Archivo > Opciones > Centro de confianza).
3. Asegúrate de tener activa la pestaña **Programador**.

## Cómo usar
1. Abre el archivo.
2. Presiona `ALT + F11` para ver el código si deseas personalizarlo.
3. Ejecuta la macro `ExportarCSV_y_GenerarImagenes`.
4. Selecciona la carpeta donde quieres que se guarden los archivos generados.
